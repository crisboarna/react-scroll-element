## Properties
<b>type</b> - Optional - Defaults to `chevron`

Type of element to be displayed:

`chevron`

`chevron-sliding`

`chevron-circle`

`chevron-circle-pulse`

<b>text</b> - Optional - Defaults to `Scroll`

Text to be displayed under element.
