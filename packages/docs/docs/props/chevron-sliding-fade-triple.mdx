# chevron-sliding-fade-triple

You can provide a `type` prop with `chevron-sliding-fade-triple`
```jsx
import ScrollElement from 'react-scroll-element';
<ScrollElement type={'chevron-sliding-fade-triple'} text={"scroll"}/>
```

import ThemeAwareComponent from "../../src/components/ThemeAwareComponent";

<ThemeAwareComponent type={'chevron-sliding-fade-triple'}/>

## With colors

You can provide `colorText` and `colorIcon` props as well.
```jsx
import ScrollElement from 'react-scroll-element';
<ScrollElement
    type={'chevron-sliding-fade-triple'}
    text={"scroll"}
    colorIcon={"#f672b0"}
    colorText={"#f672b0"}
/>
```
