# chevron-sliding-fade-spin

You can provide a `type` prop with `chevron-sliding-fade-spin`
```jsx
import ScrollElement from 'react-scroll-element';
<ScrollElement type={'chevron-sliding-fade-spin'} text={"scroll"}/>
```

import ThemeAwareComponent from "../../src/components/ThemeAwareComponent";

<ThemeAwareComponent type={'chevron-sliding-fade-spin'}/>

## With colors

You can provide `colorText` and `colorIcon` props as well.
```jsx
import ScrollElement from 'react-scroll-element';
<ScrollElement
    type={'chevron-sliding-fade-spin'}
    text={"scroll"}
    colorIcon={"#f672b0"}
    colorText={"#f672b0"}
/>
```
